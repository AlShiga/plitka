Безусловно, реализация такого проекта возможна только при поддержке настоящих профессионалов

Нашим консультантом для разработки линейки урологических тренажеров является врач-уролог, Доктор медицинских наук, Гаджиев Нариман Казиханович.

Он является одним из идейных вдохновителей данного проекта. При его непосредственной поддержке, были определены основные манипуляционные задачи при урологических вмешательствах, решить которые можно с помощью наших тренажеров.

В разработке наших тренажеров участвовал также специалист в области полимерных композиционных материалов доктор технических наук, профессор, заведующий кафедрой переработки пластмасс. Благодаря ему удалось получить тренажеры с высокой аутентичностью.
